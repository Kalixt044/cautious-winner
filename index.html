Dentro de los comandos o sentencias para SQL existen cuatro categorías:
DDL – Data Definition Language DQl
– Data Query Language DML
– Data Manipulation Language DCL
– Data Control Language
DDL o lenguaje de definición de datos en realidad consiste en los comandos SQL que se pueden usar para definir el esquema de la base de datos. Simplemente se ocupa de las descripciones del esquema de la base de datos y se usa para crear y modificar la estructura de los objetos de la base de datos en la base de datos. DDL es un conjunto de comandos SQL que se usa para crear, modificar y eliminar estructuras de la base de datos pero no datos. Estos comandos normalmente no los utiliza un usuario general, que debería acceder a la base de datos a través de una aplicación.
